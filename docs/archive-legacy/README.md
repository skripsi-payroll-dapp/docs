# Arsip — Jangan Dipakai

File di folder ini adalah versi SKPL lama (pra-refactor Gen8) yang masih mendeskripsikan
`ConfidentialCompanyVault` (FHE/Inco) dan `EmployeeLiquidityContract` ("Koperasi Karyawan") —
keduanya sudah dihapus total dari kodebase dan digantikan oleh fitur kasbon terintegrasi di
`CompanyVault`.

Dipindahkan ke sini pada 2026-07-12 karena sebelumnya berada di root proyek dengan nama
persis sama seperti dokumen resmi ("SPESIFIKASI KEBUTUHAN PERANGKAT LUNAK.docx"), sehingga
berisiko tertukar dengan dokumen yang benar.

**Dokumen SKPL/DPPL/PDHUPL yang aktif dan current ada di `payroll-web3-saas/docs/`:**
`SKPL.md` / `SKPL.docx`, `DPPL.md` / `DPPL.docx`, `PDHUPL_v2.md` / `PDHUPL_v2.docx`.
`.md` adalah sumber kebenaran; `.docx` di-generate darinya.
